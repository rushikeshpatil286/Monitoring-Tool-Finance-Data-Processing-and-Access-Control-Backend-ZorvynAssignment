# 🚀 MonitoringTool - Activity & Project Tracking System

## 📌 Overview

MonitoringTool is a web-based application developed using Java Spring Boot that helps in tracking, managing, and monitoring project activities efficiently.

The system provides a centralized dashboard where users can view project progress, manage activities, and analyze key performance indicators in a structured way.

---

## ✨ Features

### 📊 Dashboard & Activity Tracking

* Interactive dashboard displaying project metrics
* Activity Tracking Matrix for monitoring tasks
* Status tracking (To Be Planned, Planned, In Progress, Completed, Overdue)
* Search and filter functionality

### 📁 Project & Activity Management

* Create and manage projects
* Add and track multiple activities
* Assign stakeholders (Responsible, Manager, Coordinator)
* Track start and end dates

### 📤 Data Handling

* Excel upload support for bulk data
* Backend processing of uploaded data
* Structured storage and retrieval

### 🔐 Basic User Handling *(if applicable)*

* User login system
* Role-based behavior (Admin/User if implemented)
